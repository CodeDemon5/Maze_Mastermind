Isocontour Extraction from VTI Data

Using Python & VTK

‚ú® Overview

This project extracts isocontours (lines of equal scalar value) from a 2D VTI dataset using Python and VTK. The result is saved in a .vtp file which can be viewed using visualization tools like ParaView.

üìÅ Files

  • Isabel_2D.vti - Input VTI file with pressure field.
  • assignment_onecell.vtp - Output with extracted isocontour lines.
  • main.py or notebook - Core Python code using VTK.

üßê Key Concepts

  • VTK's vtkXMLImageDataReader reads .vti files.
  • Interpolation locates points where scalar values match the user-defined isovalue.
  • Each pair of points crossing the isovalue are connected to form polylines.

ü§ñ How It Works

  1. User provides an isovalue.
  2. The program checks edges of each cell to find where scalar values cross this isovalue.
  3. Interpolated points are computed and lines are formed between valid pairs.
  4. Output is saved in .vtp format.
